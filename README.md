## Description

Nest js project in development. I'll be creating an api where admin will be able to make http requests to create, update, get and delete products.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
